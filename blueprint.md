# Dinner Menu Recommendation Site - Project Blueprint

## Project Overview
A web application that recommends dinner menus based on various categories (Korean, Chinese, Japanese, Western, etc.). The application follows modern web standards using framework-less HTML, CSS, and JavaScript. It provides a fun and interactive way to decide "What should I eat for dinner?"

## Current Features (Transformed from Lotto Site)
- Random dinner menu recommendation.
- Category-based selection (implemented via menu list).
- Responsive design for mobile and web.
- Dark/Light mode support.
- Partnership inquiry form.
- Disqus comment integration.

## Planned Changes: Transformation from Lotto to Dinner Menu
### 1. UI/UX Design Updates
- **Theme**: Shift from "Lotto" (Blue/Red) to "Dinner/Food" (Orange/Red/Warm colors).
- **Icons**: Change 🎯 (Lotto) to 🍲 (Dinner) or 🍽️.
- **Visuals**: Use appetizing colors and modern UI components.

### 2. Core Functionality (main.js)

### 3. Content Updates (index.html)
